# Troubleshooting

Quick fixes for common issues when following the Copilot Studio tutorial series.

## General Issues

**"I don't have Copilot Studio access"**
- Contact your Microsoft 365 admin to enable Copilot Studio
- Ensure you have a work/school account (personal accounts won't work)

**Agent changes not appearing**
- Always click "Save" after making changes
- Refresh your test chat to see updates
- Check you're testing the correct agent version

## Episode 2: Agent Creation

**Agent not responding correctly**
- Ensure knowledge source uploaded successfully (check for green checkmark)
- Verify agent instructions are saved in Overview section

**International Rooms topic not triggering**
- Check topic trigger phrases include "international" variations
- Ensure topic is published with your agent

## Episode 3: Email Tools

**Email tools not appearing**
- Verify you have Office 365 Outlook permissions in your tenant
- Check that connectors are enabled in your environment

**Emails not sending**
- Confirm you're signed into the correct Microsoft account
- Test with your own email address first

## Still Stuck?

Open an issue in this repository with:
- Which episode you're following
- Screenshot of the error or issue
- What you expected to happen vs. what actually happened
